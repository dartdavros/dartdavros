## Обо мне

Я архитектор программного обеспечения и open source разработчик с опытом более 15 лет. Работаю с архитектурой систем, декомпозицией сложных задач, спецификациями, ADR и техническими решениями, которые можно довести до устойчивой реализации.

В последние годы активно применяю ИИ в инженерной работе: для анализа кодовой базы, подготовки спецификаций, проверки архитектурных решений, планирования изменений и сопровождения реализации. Для меня ИИ — инструмент усиления инженера, а не замена профессиональной ответственности.

Предпочитаю стартапы и небольшие команды: в таком формате важны ясная архитектура, короткий цикл принятия решений и способность быстро превращать сложные идеи в работающий продукт.
